联展精密（惠州）部件有限公司

    热烈祝贺联展精密顺利通过ISO9001质量认证体系

  • 2013-3-18

    长安福特将投6亿美元 扩大重庆基地产能

    长安福特马自达汽车宣布4月5日将投资6亿美元扩大其重庆生产基地产能。扩产后，长安福特马自达重庆工厂年产能将提升35万辆，其在中国的年总产能将超过95万辆。重庆将成为福特在密西根以外的最大生产基地。目前，长安福特马自达汽车在重庆拥有两个整车生产厂，以及一个正在建设的发动机工厂和一个变速器工厂。扩产建设项目近期将正式动工，首辆新车预计将于2014年下半年下线。
